Servicios

Diseño UX/UI

  • Interfaces de usuario atractivas y fáciles de usar, siguiendo las directrices de diseño establecidas
  • Elaborar estructuras básicas de una interfaz, como wireframes, utilizando herramientas como Figma, Sketch o Adobe XD.
  • Pruebas de usabilidad con un pequeño grupo de usuarios para identificar posibles problemas de navegación o comprensión de la interfaz.
  • Interés por aprender nuevas tecnologías y herramientas.

Desarrollo web

  • Transformar diseños gráficos (como los creados en Photoshop o Figma) en páginas web funcionales utilizando HTML, CSS y, en algunos casos, JavaScript básico.
  • Utilizar media queries y frameworks CSS como Bootstrap para garantizar una experiencia de usuario óptima en cualquier dispositivo.
  • Aplicar técnicas básicas de SEO (Search Engine Optimization) para mejorar el posicionamiento de un sitio web en los motores de búsqueda.

Creación de contenido

  • Recopilar información de los usuarios, investigar tendencias de diseño, y asistir en la creación de documentación de diseño.
